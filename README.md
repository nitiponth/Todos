## Run Locally

Clone the project

```bash
  git clone https://github.com/nitiponth/todos.git
```

Go to the project directory

```bash
  cd todos
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Frontend:** React, Mobx, Mobx-react-lite, Scss
